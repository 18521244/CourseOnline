## Yêu cầu thiết bị
    Android Studio: 4.1.2
    Phiên bản Android: 4.0 trở lên
    API: 27 trở lên
## Thư viện và công nghệ
    Backend: Java
    Giao diện: XML
    Thư viện: Retrofit, một số thư viện được Android Studio tự động thêm vào
